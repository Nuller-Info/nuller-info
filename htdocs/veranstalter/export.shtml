Wiederherstellung aus der Datenbank

Hier können Sie die Daten aus der Datenbank im Trialtool-Format herunterladen. Pro Veranstaltung gibt es eine .cfg und eine .dat - Datei. Beide Dateien müssen in das Trialtool-Verzeichnis kopiert werden (standardmäßig C:\Trialtool 3.12), danach kann die Veranstaltung im Trialtool ausgewählt und weiterbearbeitet werden.
